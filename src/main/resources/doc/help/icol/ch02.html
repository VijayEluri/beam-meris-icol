2.1. ICOL+ as BEAM plugin

As a BEAM plugin, the ICOL+ processor needs the following software to be installed in advance:

  • BEAM, version 4.8 or higher

The BEAM software can be obtained from the BEAM download page (www.brockmann-consult.de/beam).

The ICOL+ software delivery consists of two jar files, the ICOL main module and the OpenCL modules, which provide additional Java classes which are optionally used by ICOL+ for accelerating purpose in case that a proper GPU is installed in the computer:

  • beam-meris-icol-2.5.jar
  • opencl_jars.zip

The beam-meris-icol-2.5.jar files needs to be copied into the 'modules' folder of the BEAM installation (Figure 2.1). The opencl_jars.zip files needs to be unpacked into the 'lib' folder of the BEAM installation. Note that existing versions of ICOL jar files (e.g. ICOL1.0) need to be replaced/overwritten to avoid version conflicts of Java classes.
